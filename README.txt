2. HOW TO RUN THE GAME FROM CODE
Open the terminal in this directory and execute the command:
                    python agent.py
3. FEATURES
The game includes a graphical menu with the following selection modes:

Mode 1: Watch AI Play

Statement: This mode allows observation of the AI using a model that has been trained (by the student) for over 1000 games.

Mode 2: Train the AI

Statement: This mode enables continued AI learning/training.

Mode 3: Versus Mode (PvAI)

Statement: This is the competitive mode (Player vs. AI).
